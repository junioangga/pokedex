## **INTRODUCTION**

A working pokedex that can separate pokemon by its types, favorite pokemon, and see pokemon details.


### **Framework and Tools**
Vue.js used to make this project

### **Dependencies**
- [**bulma**](https://bulma.io/): **bulma** is a really flexible and useful css framework that fasten development process. It also already support vue 3.
- [**axios**](https://axios-http.com/docs/intro): **axios** is an excelent and efficient tool to fetch REST API

### **Running the project**
This project can be run locally my typing "npm run serve" in your console emulator
